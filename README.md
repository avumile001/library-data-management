# library-data-management

## About the Project

This project is a simple library data management system built using Python and MySQL. It focuses on creating a basic ETL (Extract, Transform, Load) pipeline that takes data from CSV files, processes it, and loads it into a database for storage and analysis.

## What I Used
Python
MySQL
Pandas
SQLAlchemy
PyMySQL
Git & GitHub

## What I Learned
How to build a simple ETL pipeline
How to connect Python to a MySQL database
How to read and process CSV files using Pandas
How to load data into a relational database
Basic database management and SQL concepts
How to use Git and GitHub for version control

## How It Works
Extract data from CSV files (books, members, loans)
Clean and prepare the data using Python
Load the cleaned data into a MySQL database
Store and manage data for future use and analysis

## Purpose of the Project
The purpose of this project is to practice data engineering fundamentals and understand how data moves from raw files into a structured database system.
